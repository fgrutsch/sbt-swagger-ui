# sbt-swagger-ui

SBT plugin that generates a swagger-ui site for static website hosting.

## Getting Started

Add the following dependency to your `project/plugins.sbt`:

```scala
addSbtPlugin("com.fgrutsch" % "sbt-swagger-ui" % "x.x.x")
```
